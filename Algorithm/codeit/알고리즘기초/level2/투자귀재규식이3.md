# 투자의 귀재 규식이3



## 문제

> 주어진 리스트의 구간의 최대합을 구하는 문제
>
> [7, -3, 4, -8] 의 경우, 8을 리턴해야 한다.
>
> 시간복잡도 O(n) 으로 구현





## 풀이법 및 코드

```
[7, -3, 4, 8] 의 경우

7부터 시작하여
7에서 -3을 더한 경우와 -3을 비교
4가 -3 보다 더 크므로
7과 4를 비교 => 7이 더 큼
따라서 7이 [7, -3] 구간에서 최대

다음 회에
앞서 7에서 -3을 한 결과값 4에서 2번째 요소 4를 더한 경우와 2번째 요소 4를 비교
(7에서 -3을 한 결과값에서 4룰 더하는 이유? : [7, -3, 4] 까지 새로 들어올 2번째 요소까지의 누적합과 2번째 요소 자체를 비교)
새로 들어올 2번째 요소까지의 누적합 8이 4보다 크므로
[7, -3] 구간의 최대인 7과 8을 비교
=> [7, -3, 4] 구간의 최대는 8

마지막 회에
앞서 구했던 누적합 8과 새로 들어올 3번째 요소 -8를 더한 값과 -8을 비교
0이 더 크므로 [7, -3, 4] 구간의 최대 8과 0을 비교
=> [7, -3, 4, -8]의 최대 구간은 8

따라서 답은 8
```



```python
def sublist_max(profits):
    max_profit_now = profits[0] # 현재까지의 최대 수익
    max_check = profits[0] # 새로운 요소가 들어올 때 최대 수익을 정하기 위한 변수
    
    for idx in range(1, len(profits)):
        max_check = max(max_check + profits[idx], profits[idx])
        max_profit_now = max(max_profit_now, max_check)
    
    return max_profit_now
    


# 테스트
print(sublist_max([7, -3, 4, -8]))
print(sublist_max([-2, -3, 4, -1, -2, 1, 5, -3, -1]))

```
